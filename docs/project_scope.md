# Project Scope – Café Bar Momentos Felices

## 1. Visión del Proyecto

"Café Bar Momentos Felices" es una aplicación web informativa diseñada para representar digitalmente un café-bar local. 

El objetivo principal es ofrecer a los clientes una experiencia previa a su visita, permitiéndoles conocer el menú, el ambiente del establecimiento y los canales de contacto.

La aplicación busca fortalecer la presencia digital del negocio y mejorar su posicionamiento como marca local.

---

## 2. Problema que Resuelve

Muchos pequeños negocios no cuentan con una presencia web profesional, lo que limita su visibilidad y capacidad de atraer nuevos clientes.

Esta aplicación resuelve ese problema proporcionando:

- Información clara y accesible del menú.
- Imagen visual atractiva del negocio.
- Datos de contacto organizados.
- Experiencia moderna y adaptable a dispositivos móviles.

---

## 3. Usuario Objetivo

La aplicación está dirigida a:

- Personas que buscan un lugar agradable para compartir con amigos o familia.
- Clientes interesados en café, bebidas especiales y postres artesanales.
- Usuarios que desean consultar el menú antes de visitar el establecimiento.
- Público joven y adulto con acceso a dispositivos móviles.

---

## 4. Funcionalidades Principales (Core Features)

### 4.1 Sección de Inicio (Home)
- Imagen principal representativa del café-bar.
- Mensaje de bienvenida.
- Llamado a la acción (Call To Action – CTA) para visualizar el menú.

### 4.2 Sección de Menú
- Productos organizados por categorías:
  - Cafés
  - Bebidas
  - Postres
- Nombre del producto.
- Precio.
- Breve descripción.
- Imagen ilustrativa.

### 4.3 Sección Sobre Nosotros
- Descripción del concepto del negocio.
- Misión y propuesta de valor.
- Ambiente y experiencia ofrecida.

### 4.4 Sección de Contacto
- Dirección física.
- Número telefónico.
- Correo electrónico.
- Horario de atención.

---

## 5. Alcance del Proyecto (Scope)

### Versión 1.0
- Aplicación web estática desarrollada con HTML5 y CSS3.
- Diseño responsive adaptable a dispositivos móviles.
- Estructura semántica organizada.
- Uso de imágenes optimizadas.

### Fuera del Alcance (Por ahora)
- Sistema de autenticación de usuarios.
- Carrito de compras.
- Base de datos.
- Procesamiento de pagos.

---

## 6. Escalabilidad Futura

En futuras versiones se contempla:

- Integración de JavaScript para interactividad.
- Implementación con React para una arquitectura moderna.
- Conexión con Firebase para base de datos y autenticación.
- Sistema de pedidos en línea.
