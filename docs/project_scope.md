# Project Scope - Café Bar Momentos Felices

## Descripción General
"Café Bar Momentos Felices" es una aplicación web informativa diseñada para promocionar un café-bar local. Su objetivo es presentar de manera atractiva el menú, el ambiente del lugar y la información de contacto, permitiendo que los clientes conozcan los productos y servicios ofrecidos.

## Usuario Objetivo
- Personas que buscan un lugar agradable para compartir con amigos o familia.
- Clientes locales interesados en café, bebidas especiales y postres.
- Visitantes que desean consultar el menú antes de asistir al establecimiento.

## Funcionalidades Principales (Core Features)

1. **Sección de Inicio (Home):**
   - Imagen principal llamativa.
   - Mensaje de bienvenida.
   - Botón de llamada a la acción (Ver menú).

2. **Sección de Menú:**
   - Lista de productos organizados por categorías (Cafés, Bebidas, Postres).
   - Nombre del producto.
   - Precio.
   - Breve descripción.

3. **Sección Sobre Nosotros:**
   - Información del negocio.
   - Historia o misión del café-bar.

4. **Sección de Contacto:**
   - Dirección.
   - Número telefónico.
   - Horario de atención.

## Reglas del Proyecto

- La primera versión será estática (HTML y CSS).
- El diseño debe ser adaptable a dispositivos móviles (Responsive Design).
- No requiere sistema de usuarios en esta fase.
- En futuras fases se agregará interactividad con JavaScript.
