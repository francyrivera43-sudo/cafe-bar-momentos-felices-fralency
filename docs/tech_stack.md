# Tech Stack – Café Bar Momentos Felices

## 1. Tecnologías Actuales (Versión 1.0)

### HTML5
- Utilizado para la estructura semántica del sitio web.
- Permite organizar el contenido en secciones como:
  - Header
  - Menú
  - Sobre Nosotros
  - Contacto
- Mejora la accesibilidad y el SEO básico.

### CSS3
- Encargado del diseño visual y la presentación.
- Implementa:
  - Diseño responsive (Mobile First)
  - Sistema de colores definido en el Design System
  - Estilos para botones, tarjetas y secciones
  - Espaciado consistente

### Git
- Sistema de control de versiones.
- Permite registrar cambios del proyecto.
- Facilita el trabajo organizado y profesional.

### GitHub
- Repositorio remoto del proyecto.
- Almacenamiento en la nube.
- Seguimiento de versiones y respaldo del código.

---

## 2. Tecnologías Futuras (Escalabilidad)

### JavaScript
- Agregará interactividad al sitio.
- Posibles funcionalidades:
  - Menú dinámico
  - Animaciones
  - Validación de formularios

### React
- Framework para construir interfaces modernas.
- Permitirá:
  - Componentización
  - Mejor rendimiento
  - Escalabilidad del proyecto

### Firebase (Planificado)
- Base de datos en la nube.
- Posible integración para:
  - Autenticación de usuarios
  - Sistema de pedidos
  - Almacenamiento de datos
